About AmkyawDev Editor

A powerful, lightweight, and modern web-based code editor for developers.

Our Mission

AmkyawDev Editor is designed to provide a seamless development experience directly in your browser. Whether you're a beginner or an experienced developer, our tool offers the features you need to build and test your web projects efficiently.

Key Features

  • Project Management: Easily create, manage, and delete multiple web projects.
  • Multi-File Editing: Edit HTML, CSS, and JavaScript files with a clean, intuitive interface.
  • Live Preview: See your changes in real-time with our integrated preview system.
  • Modern UI: A sleek, responsive design that works across different devices.
  • File System: Advanced file management for organizing your project assets.

Technology Stack

Built with modern web technologies to ensure performance and reliability:

  • Backend: Flask (Python)
  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • Styling: Custom CSS with 3D transforms and smooth transitions
  • API: RESTful API for file and project operations
